# Shop

Progression blueprints assume that the following upgrades are purchased at the end of the listed milestone

## Milestone 1

### After
Should have `6000` Research Credits (RC) available from all jobs

Purchase:
* Platform Capacity 1
* Cutter
* Trash
* Reverse Rotator
* 180 Rotator
* Overflow Splitter
* Wire Basics
* Label

## Milestone 2 (Fluids)

### During 
* Swapper
* Vortext Inputs 1
* Vortext Inputs 2


### After
Should have `~7700` RC available from all jobs and a few operator levels

* 3rd machine level
* Longer Platforms
* Bent Stacker

## Pre-Milestone 3
Set up Milestone 1 and 2 shape dumps for operator level. This will farm plenty of RC

Purchase:
* Vortext Inputs 3
* Vortext Inputs 4
* Anything else using Milestone shape RC

### Milestone 3
* Fluid wagons
* 4+ train capacity
* All train colors
